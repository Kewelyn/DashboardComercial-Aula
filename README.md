# Dashboard Comercial - Power BI

## Sobre o Projeto
Este dashboard foi desenvolvido durante uma aula prática de Power BI, com o objetivo de aplicar conceitos de modelagem de dados, 
criação de métricas, visualização de informações e construção de painéis interativos.

## Tecnologias Utilizadas
- Power BI
- Power Query
- DAX
